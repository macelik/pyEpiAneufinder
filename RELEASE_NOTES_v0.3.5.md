## pyEpiAneufinder 0.3.5

This release establishes the first package-ready foundation for
`pyEpiAneufinder`.

### Highlights

- PyPI package metadata cleaned up
- Minimum dependency versions aligned with Python >= 3.10
- README streamlined as a package front page
- Sphinx documentation expanded and organized for Read the Docs
- Quickstart split into linked workflow pages with figures
- Separate GitHub Actions workflows for tests and docs
- Public API tests added for documented package behavior
- GPL license metadata and LICENSE file added
